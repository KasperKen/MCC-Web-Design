HTML Tags

I have always thought HTML was a coding language. However this isn't the case. HTML is markup language which only exists to format documents in a way that a web browser can read and display. In simpler terms it is a formatting language. Each tag used is merely a way for the browser to format particular text or images into human readable blocks. They also allow web crawlers, and assistive technologies to better parse through the information.

For example the <head> tag containes information important to the document such as the meta title information, each of which would be it's own tag whithin the header brackets. The <title> tag for example contains the title of the webpage to be crawled by search engines, and be displayed in the title bar.

The <body> tag contains all the information which will be displayed in the body of the website. This is the actual web content such headings and paragraphes which can be broken into their own tags <h1>, <h2>, and <p>. Another important thing to note is that all tags need an opening and closing tag. To open a tag you would use brackets to declare a tag, and then brackets with a forward slash to indicate that you are closing the tag.

For Example:

<p>
This is a paragraph
</p>

Some other examples of common tags may include:

  • <hr> creates a horizontal line to create a break in the page, such as the one above this text.
  • <br> creates line breaks thus breaking the text into more human readable blocks on the display.
  • <ul> creates an unordered list, such as the one being used here
  • <ol> creates an ordered list
  • <li> structures content in a list within a ul, or ol element
