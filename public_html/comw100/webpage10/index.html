Transform#3
The image below begins scaled down to half size using scale(.5). The image has alsa had a border radius of 25% applied to it. It will transform back to just over normal scale while holding a click using scale(1.4, 1). It starts at a larger size because the table row would otherwise resize the table row. I'm sure I could have set the row's size in the html, but this accomplishes the same thing using transformations instead. I've used transform .8s to give the transformations a short animation effect.
